大连理工大学软件学院绩点计算系统
========
######&emsp;**本系统**是基于Python的爬虫库对大连理工大学的URP教务系统信息进行爬取，通过得到用户的学号和密码，来模拟登陆教务系统，进而对教务系统内的成绩进行抓取并执行相关的计算。整个过程没有对教务系统进行任何的修改和破坏，同时也不会记录或上传同学们的学号和密码。
---
## 使用方法
#### 使用方法1(有Python环境的条件下):
	1. 下载 zip 包，解压
	2. 在解压目录下直接执行 python spider.py

#### 使用方法2(win平台下即可):
	直接下载dist下的spider文件执行
